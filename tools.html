Additional tool installations¶

Run the --install-<toolname> for each of the tools below. Each is installed (or symlinked) into ~/opt/bin, so you’ll want to make sure that’s on your path.

Note

Some of these tools are available in conda. In those cases, they are “installed” as follows:

  • create a conda environment named after the tool

  • symlink the tool’s executable from the conda environment over to ~/opt/bin

This means that you do not have to conda activate an environment to use an installed tool. As long as ~/opt/bin is on your path, you can use it immediately. This allows for more modular updating of the tools and avoiding conflicting dependencies.

icdiff¶

./setup.sh --install-icdiff

icdiff makes an easy-to-read, side-by-side colored diff between files. It’s used for ./setup.sh --diffs.

fzf¶

./setup.sh --install-fzf

fzf is a fuzzy-finder interface that works with stdin. It integrates with bash so that when you use Ctrl-R (the standard bash way of reverse-search through history), you’ll instead get the fzf interface.

ripgrep (rg)¶

./setup.sh --install-ripgrep

ripgrep is a fast code-searching tool. It is like grep, but by default skips files in .gitignore, binary files, and hidden files.

fd¶

./setup.sh --install-fd

fd is a much faster and more ergonomic find.

visidata (vd)¶

./setup.sh --install-vd

visidata ia a powerful spreadsheet-like tool for viewing, sorting, searching, and manipulating data directly in the terminal. Any files that pandas can open, visidata can open too.

black¶

./setup.sh --install-black

black reformats Python files to conform to PEP8 style conventions.

radian¶

./setup.sh --install-radian

radian is a replacement shell for R. It has syntax highlighting, multiline editings, and tab completion built in. can be used with any version of R, in a conda environment or otherwise.

git-cola¶

./setup.sh --install-git-cola

git-cola is a graphical tool for incrementally making git commits. Very useful, for example, at the end of a coding session and you want to make atomic commits from all the changes you made.

bat¶

./setup.sh --install-bat

bat is a replacement for cat, with syntax highlighting, line numbers, non-printable characters, and git diffs.

alacritty¶

./setup.sh --install-alacritty

alacritty is a cross-platform GPU-accelerated terminal emulator. The speed is most noticable when catting large files over tmux.

jq¶

./setup.sh --install-jq

jq <https://stedolan.github.io/jq/> is like sed or awk for JSON data.

docker¶

./setup.sh --install-docker

docker runs containers. Needs root access, and the installation here is currently only supported on Linux.
